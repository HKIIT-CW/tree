                         AVL 旋轉原理解析
                    

                    
                    

                        
1. 單一右旋 (LL Rotation)

                        

                            
觸發時機： 節點的 左子樹的左子樹 插入了新節點，導致平衡因子大於 1。

                            

                                  A (BF=2)

                                 / 

                                B (BF=1)    ==右旋 A==>    B

                                /                             / \

                               C                            C   A
                            

                        

                    


                    

                        
2. 單一左旋 (RR Rotation)

                        

                            
觸發時機： 節點的 右子樹的右子樹 插入了新節點，導致平衡因子小於 -1。

                            

                                A (BF=-2)

                                 \

                                  B (BF=-1)   ==左旋 A==>    B

                                   \                            / \

                                    C                          A   C
